О переходе на новый почтовый сервер
Уважаемый(ая) %USERNAME%!
Обратите внимание! Все кто получил первое письмо содержало ошибочные данные
На этой неделе будет осуществлен переход на новый корпоративный почтовый сервер. Точная дата будет сообщена дополнительным письмом. В данное время сервер работает в режиме тестовой эксплуатации.
Доступ к почтовому серверу работает через веб-браузер по адресу: https://zimbra.voel.ru. Так же для доступа к серверу можно использовать ярлык на рабочем столе "ПОЧТА ВОЭ (Zimbra)".
Старые письма останутся в почтовой программе Outlook, далее будет осуществлен перенос писем на новый сервер.
Инструкция по работе с почтой

Для входа используйте:

Имя пользователя: %LOGIN%

Пароль: %PASS%

Обязательно запишите эти данные для входа в надежное место!

Напоминаем, что по правилам корпоративной политики, пароли нельзя хранить в открытом доступе например на листочке приклееном к монитору и т.п.
